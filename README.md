# Flask-vite-RTS exercise

this is a exercise of a fullstack usage example.

## including

Frontend: node, React(UI library), vite(Build toolchain), SWC(compiler);

Backend: Flask, flask_cors, Flask-WTF.

## Requirements

1. you can simply using venv as your virtual python environment and run `pip install -r requirments`;

2. then go to rts-vite-frontend and run `npm i`.
